# Student Housing Platform

A web-based platform designed to help university students find suitable housing near campus and help landlords publish accommodation listings in a simple and organized way.

## Team Members
- Theresa
- Celine
- Pamela
- Agnes

## Project Overview
The LU Student Housing Platform is a university-focused housing website where:
- students can browse and search available housing listings
- landlords can post and manage accommodation listings
- admins can review and approve submitted listings

The goal of this project is to make student housing search easier, more trustworthy, and more accessible.

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB 

## Project Management Approach
This project follows the Scrum methodology.
We use:
- Jira for backlog and sprint management
- GitHub for version control and collaboration
- Planning Poker for user story estimation

## Branching Strategy
This repository uses a simple team-based branching workflow:

- `main` → contains the latest stable and integrated version of the project
- `theresa` → Theresa's working branch
- `celine` → Celine's working branch
- `pamela` → Pamela's working branch
- `agnes` → Agnes's working branch

Each team member works on her own branch and merges completed work into `main` through pull requests.

## Workflow Rules
- Each team member develops features on her own branch
- Direct pushes of unfinished work to `main` are avoided
- Pull requests are used before merging into `main`
- Team members pull the latest `main` before starting new work
- Commit messages should be clear and meaningful
